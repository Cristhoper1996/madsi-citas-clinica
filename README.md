# Sistema de Gestión de Citas para una Clínica

## Descripción del sistema
El sistema de gestión de citas para una clínica es una aplicación diseñada para administrar de forma eficiente las citas médicas, permitiendo organizar los horarios de atención y mantener un mejor control de pacientes y médicos.

## Objetivo del sistema
El objetivo del sistema es resolver los problemas de desorganización y errores en la asignación de citas, facilitando la programación, modificación y cancelación de citas para beneficio del personal administrativo, médicos y pacientes.

## Actores del sistema
- Paciente  
- Recepcionista  
- Médico  

## Funcionalidades principales
- Registrar citas médicas  
- Cancelar o modificar citas  
- Consultar la agenda médica  
- Ver disponibilidad de horarios
